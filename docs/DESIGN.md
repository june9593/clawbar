# ClawBar — UI/UX Design Specification

> **角色**: Designer
> **版本**: v1.0
> **日期**: 2026-03-31

---

## 1. 设计原则

1. **macOS 原生感**: 遵循 Apple HIG，使用 SF 字体、vibrancy 效果、系统配色
2. **轻量不打扰**: 小窗口、快速唤起/隐藏，不遮挡工作区
3. **信息密度适中**: 消息气泡紧凑但可读，留白克制
4. **无障碍**: 支持键盘导航、高对比度、合理字号

---

## 2. Menu Bar 图标

### 2.1 图标规格
- 格式: PNG Template Image（macOS 自动适配亮/暗 menu bar）
- 命名: `iconTemplate.png` + `iconTemplate@2x.png`
- 尺寸: 16x16 pt (32x32 px @2x)
- 设计: 龙虾爪简化轮廓，线条粗细 1.5pt
- 颜色: 纯黑（Template Image 由系统着色）

### 2.2 状态变化
| 状态 | 图标表现 |
|------|---------|
| 已连接 | 正常图标 |
| 已断开 | 图标 + 红色小圆点（右上角 badge） |
| 有新消息 | 图标闪烁一次（P2 功能） |

## 3. 聊天窗口布局

### 3.1 整体结构

```
┌──────────────────────────────────┐
│  Title Bar (40px)                │  ← 可拖拽区域
│  [≡ Agent ▼] [session] [📌] [⚙] │
├──────────────────────────────────┤
│                                  │
│  Message List (flex: 1)          │  ← 滚动区域
│                                  │
│  ┌─────────────────────────────┐ │
│  │ 🤖 AI 消息气泡 (左对齐)      │ │
│  └─────────────────────────────┘ │
│                                  │
│         ┌──────────────────────┐ │
│         │ 👤 User 消息 (右对齐) │ │
│         └──────────────────────┘ │
│                                  │
├──────────────────────────────────┤
│  Input Area (52-150px)           │
│  ┌────────────────────────┐ [↑]  │
│  │ 输入消息...              │     │
│  └────────────────────────┘      │
└──────────────────────────────────┘
```

### 3.2 Title Bar（40px 高度）

**布局**: `[会话切换按钮] ─── [会话名称] ─── [Pin] [设置]`

- **左侧**: Agent 图标 + 下拉箭头（点击打开会话切换器）
- **中间**: 当前 Agent 名称 / 会话名称（居中，字号 13px，semibold）
- **右侧**: Pin 按钮 + 设置齿轮按钮
- **整个 Title Bar**: `-webkit-app-region: drag`（可拖拽）
- **按钮区域**: `-webkit-app-region: no-drag`
- **背景**: 毛玻璃效果 `backdrop-filter: blur(20px) saturate(180%)`

**Pin 按钮**:
- 未 pin: 📌 灰色 → hover 变深
- 已 pin: 📌 蓝色（填充状态），tooltip "取消置顶"

### 3.3 Message List（弹性高度）

**滚动行为**:
- 默认锁定底部（新消息自动滚动）
- 用户向上滚动时解锁，显示 "↓ 回到底部" 按钮
- 触达底部时重新锁定

**消息气泡**:

#### User 消息（右对齐）
```
                        ┌────────────────┐
                        │ 消息内容        │ ← bg: userBubble, color: white
                        │                │    border-radius: 18px 18px 4px 18px
                        └────────────────┘
                               12:34 PM    ← 时间戳，tertiary 色，11px
```

#### AI 消息（左对齐）
```
┌─────────────────────────────┐
│ 🦞                           │ ← Agent 头像（emoji 或图标）
│ Markdown 渲染内容             │ ← bg: assistantBubble
│                              │    border-radius: 18px 18px 18px 4px
│ ```python                    │
│ def hello():                 │ ← 代码块：mono 字体，暗色背景
│     print("world")           │    右上角 "Copy" 按钮
│ ```                          │
└─────────────────────────────┘
12:34 PM                         ← 时间戳
```

**气泡间距**: 同发送者连续消息间距 4px，不同发送者间距 16px

#### Typing 指示器
```
┌──────────────┐
│ 🦞 ● ● ●     │ ← 三个圆点依次跳动动画
└──────────────┘
```

### 3.4 Input Area（52-150px）

```
┌─────────────────────────────────────┐
│ ┌─────────────────────────────┐ [↑] │
│ │ 给 Agent 发送消息...          │     │ ← placeholder
│ │                              │     │    auto-resize textarea
│ └─────────────────────────────┘     │
└─────────────────────────────────────┘
```

- **输入框**: 无边框 textarea，自动高度（1-5 行）
- **发送按钮**: 圆形箭头 ↑，只在有内容时显示蓝色可点击状态
- **快捷键**: Enter = 发送，Shift+Enter = 换行
- **底部圆角**: border-radius: 0 0 12px 12px

## 6. 状态设计

### 6.1 空状态（无会话）
```
┌──────────────────────────────────┐
│           🦞                      │
│     欢迎使用 ClawBar              │
│   连接你的 OpenClaw 实例          │
│   开始你的第一次对话              │
│                                  │
│   [开始配置]                      │
└──────────────────────────────────┘
```

### 6.2 加载状态
- 首次加载: 整体骨架屏（shimmer 效果）
- 发送消息: 输入框禁用 + 打字指示器
- 切换会话: Message List 淡入淡出

### 6.3 错误状态
```
┌──────────────────────────────────┐
│   ⚠️ 无法连接到 OpenClaw         │
│                                  │
│   请检查 openclaw CLI 是否已安装  │
│   并确保服务正在运行              │
│                                  │
│   [重试]  [打开设置]              │
└──────────────────────────────────┘
```

### 6.4 断开连接状态
- 标题栏显示 "⚠ 已断开" 标签（红色）
- 输入框 placeholder 变为 "连接断开，无法发送..."
- 输入框禁用

## 7. 窗口行为

### 7.1 弹出动画
- 方向: 从 menu bar 图标位置向下展开
- 动画: `transform: translateY(-8px) → translateY(0)` + `opacity: 0 → 1`
- 时长: 200ms，easing: easeOut

### 7.2 隐藏动画
- 动画: `opacity: 1 → 0` + `transform: translateY(0) → translateY(-8px)`
- 时长: 150ms

### 7.3 Resize 行为
- 可从四边和四角 resize
- resize 时显示窗口尺寸提示
- 松手后尺寸保存到 electron-store，下次启动恢复

### 7.4 拖拽行为
- 仅标题栏区域可拖拽
- 拖拽后窗口位置保存，下次显示时恢复到上次位置
- 如果上次位置超出屏幕（如外接显示器移除），自动重置到 menu bar 图标下方

## 8. 交互规范

| 交互 | 行为 |
|------|------|
| 点击 Tray 图标 | Toggle 窗口显示/隐藏 |
| ESC | 隐藏窗口（非 Pin 状态） |
| Cmd+W | 隐藏窗口（非 Pin 状态） |
| Enter | 发送消息 |
| Shift+Enter | 输入换行 |
| Cmd+N | 新建会话 |
| Cmd+, | 打开设置 |
| 点击窗口外部 | 可选隐藏窗口（设置项，默认关闭） |
| 滚轮 | 滚动消息列表 |
| 右键消息 | 复制 / 重试 / 删除（上下文菜单） |

## 10. Design Token 引用

所有颜色、字号、间距、圆角、阴影值见 `docs/design-tokens.json`。

开发时通过 CSS 自定义属性应用：
```css
:root {
  --color-bg-primary: #FFFFFF;
  --color-text-primary: #1D1D1F;
  --radius-bubble: 18px;
  /* ... */
}

[data-theme="dark"] {
  --color-bg-primary: #1C1C1E;
  --color-text-primary: #F5F5F7;
  /* ... */
}
```

组件中禁止硬编码颜色值，全部使用 CSS 变量。
